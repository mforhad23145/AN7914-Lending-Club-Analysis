## Project Overview
This project analyses the determinants of interest rates in the consumer 
credit market using data from Lending Club, a peer-to-peer lending platform. 
The dataset contains 10,000 loan observations. Five OLS regression models 
are estimated to identify key predictors of interest rates.

## Files
- `Md_Mahmudul_Hoque.ipynb` — Main analysis notebook (Python)
- `loans_dataset.csv` — Lending Club loans dataset (10,000 observations)

## Requirements
